*How to run this bot*
- `git clone https://github.com/animespotflix/kanaobot`
- `cd kanaobot`
- `pip3 install -r requirements.txt`
- `cp sample_config.py config.py` and satisfy the values in `config.py`
- `python3 -m kanarobot`
